# Patio Kitchen & Bar - Static Website

Welcome to the official website project for **Patio Kitchen & Bar** in Da Nang!

This is a simple static website built to showcase the vibe, food, and spirit of Patio. It features a clean design with an emphasis on beautiful imagery, easy navigation, and smooth mobile responsiveness.

## Project Details
- Built with: HTML5, CSS3, a little JavaScript
- Hosting: Static hosting (Netlify, Vercel, or similar recommended)
- Images: A mix of original Patio photos and curated images from Unsplash (licensed for free commercial use)
